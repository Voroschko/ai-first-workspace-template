# Дизайн проекта Gofer

Папка для хранения всех дизайн-материалов.

## Референсы и стилистика

- **`style-reference.md`** - Стилистика для лендинга (основана на https://ru.sfer.ai/)
- **`examples/sfer-ai-reference.md`** - Детальный анализ референс-сайта

## Что здесь должно быть:

- Скриншоты сайтов-примеров
- Дизайн-макеты (если есть)
- Цветовая палитра
- Шрифты и типографика
- Логотипы и брендинг
- Иконки и графические элементы

## Формат

- Изображения: `.png`, `.jpg`, `.svg`
- Макеты: `.fig`, `.sketch`, `.xd` (если есть)
- Описания дизайна: `.md` файлы с описаниями

## Референс-сайт

**Основной референс:** https://ru.sfer.ai/
- Выбран как основа для стилистики лендинга Go Offer
- Стилистика будет адаптирована под бренд Go Offer
