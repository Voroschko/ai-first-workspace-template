# Ссылки и ресурсы

Папка для хранения внешних ссылок и ресурсов, связанных с проектом.

## Что здесь должно быть:

- Ссылки на полезные ресурсы
- Документация библиотек и инструментов
- Статьи и материалы для вдохновения
- Ресурсы для разработки

## Формат

Создайте файл `links.md` со списком ссылок с описаниями, или отдельные файлы для разных категорий ресурсов.
