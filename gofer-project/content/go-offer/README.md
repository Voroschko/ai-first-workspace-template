# Контент Go Offer

Папка с контентом для проекта Go Offer, который будет использоваться на сайте.

## Файлы

- **`gooffer-base-complete.md`** ⭐ **[CANONICAL]** - Основная база знаний о продукте Go Offer
- **`services-detailed.md`** - Детальное описание всех услуг с ссылками на Notion
- **`services.md`** - Краткая информация об услугах
- **`services-full.md`** - Полное описание всех услуг
- **`pricing.md`** - Тарифные планы и цены
- **`about.md`** - Информация о компании и команде

## Источник информации

- **Основной сайт:** https://go-offer.us/
- **База знаний:** goofferbace.txt (сохранена в gooffer-base-complete.md)

## ⚠️ ВАЖНО: Использование информации

**При работе над сайтом и создании контента:**
1. **Всегда ссылайтесь** на `gooffer-base-complete.md` как на [CANONICAL] источник
2. **Используйте информацию** из этого файла для описания услуг, преимуществ, результатов
3. **Ссылайтесь на детальные услуги** из `services-detailed.md` для конкретных описаний
4. **Обновляйте информацию** в этих файлах при получении новых данных
