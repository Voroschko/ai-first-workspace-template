# Контент для сайта

Папка для хранения контента, который будет использоваться на сайте.

## Структура

- **`go-offer/`** - Контент проекта Go Offer
  - `services.md` - Информация об услугах

## Что здесь должно быть:

- Тексты страниц
- Описания услуг/продуктов
- Изображения для контента
- Видео и медиа-файлы
- FAQ и часто задаваемые вопросы

## Формат

- Тексты: `.md` файлы
- Изображения: подпапки по типам контента
- Медиа: соответствующие форматы

## Проекты

### Go Offer

Контент для проекта Go Offer находится в папке `go-offer/`.

- Источник: Notion (https://www.notion.so/Go-Offer-101329f89df2800992fdc5964bcaa5a7)
- Услуги: `go-offer/services.md`
