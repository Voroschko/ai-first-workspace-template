# Инструкции по извлечению контента из Notion

Для автоматического извлечения контента из Notion страницы Go-Offer необходимо выполнить следующие шаги:

## Вариант 1: Использование Notion API (Автоматический)

### Шаг 1: Создать интеграцию в Notion

1. Перейдите на https://www.notion.so/my-integrations
2. Нажмите "New integration" или "Новая интеграция"
3. Укажите название, например "Gofer Project Integration"
4. Выберите workspace, где находится страница Go-Offer
5. Скопируйте **Internal Integration Token**

### Шаг 2: Предоставить доступ интеграции к странице

1. Откройте страницу Go-Offer в Notion: https://www.notion.so/Go-Offer-101329f89df2800992fdc5964bcaa5a7
2. Нажмите на "..." в правом верхнем углу
3. Выберите "Add connections" или "Добавить связи"
4. Найдите вашу интеграцию и добавьте её
5. Также добавьте доступ ко всем дочерним страницам

### Шаг 3: Установить токен (Windows PowerShell)

```powershell
$env:NOTION_TOKEN="ваш_токен_здесь"
```

Или для постоянной установки:

```powershell
[System.Environment]::SetEnvironmentVariable('NOTION_TOKEN', 'ваш_токен_здесь', 'User')
```

### Шаг 4: Установить зависимости

```powershell
pip install requests
```

### Шаг 5: Запустить скрипт

```powershell
cd gofer-project
python fetch-notion-content.py
```

## Вариант 2: Ручной экспорт (Альтернативный)

Если вы не хотите использовать API, вы можете:

1. **Экспортировать из Notion:**
   - Откройте страницу Go-Offer
   - Нажмите "..." → "Export"
   - Выберите формат "Markdown & CSV"
   - Скачайте файлы

2. **Или скопировать контент вручную:**
   - Скопируйте содержимое страницы
   - Сохраните в файлы в соответствующих папках:
     - `gofer-project/info/` - основная информация
     - `gofer-project/content/` - контент для сайта
     - `gofer-project/design/` - дизайн-материалы

## Вариант 3: Предоставить доступ к странице

Если страница публичная или вы предоставите мне доступ другим способом, я смогу извлечь информацию автоматически.

---

**Важно:** Скрипт `fetch-notion-content.py` автоматически:
- Извлечет все блоки со страницы
- Обработает дочерние страницы
- Сохранит контент в markdown формате в `gofer-project/info/notion-content.md`
