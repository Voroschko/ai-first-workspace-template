# Проверка сборки перед деплоем

## Быстрая проверка:

```bash
cd Website
npm install
npm run build
```

Если сборка падает локально, она упадет и на Vercel.

## Частые проблемы:

1. **Отсутствующие файлы** - убедитесь, что все файлы закоммичены в Git
2. **Проблемы с путями** - проверьте импорты `@/components/...`
3. **TypeScript ошибки** - запустите `npx tsc --noEmit`
4. **Отсутствующие зависимости** - проверьте `package.json`
