# Быстрая инструкция по установке (обход проблем с PowerShell)

## ✅ Самый простой способ установить Node.js

### Шаг 1: Скачать через браузер

1. Откройте ваш браузер (Chrome, Edge, Firefox и т.д.)
2. Перейдите на: **https://nodejs.org/**
3. Нажмите большую зеленую кнопку **"Download Node.js (LTS)"**
4. Дождитесь завершения загрузки

### Шаг 2: Установить

1. Откройте папку **"Загрузки"** (Downloads)
2. Найдите файл `node-v20.x.x-x64.msi` (или похожий)
3. **Дважды кликните** по файлу
4. Следуйте инструкциям мастера установки:
   - Нажимайте "Next"
   - Примите условия
   - **Убедитесь, что выбрана опция "Add to PATH"**
   - Нажмите "Install"
5. Дождитесь завершения установки

### Шаг 3: Проверить установку

1. Закройте все окна терминала
2. Откройте **новый** PowerShell или Command Prompt
3. Выполните:

```powershell
node --version
```

Должна появиться версия, например: `v20.11.0`

```powershell
npm --version
```

Должна появиться версия npm, например: `10.2.4`

### Шаг 4: Установить зависимости проекта

```powershell
cd C:\Users\xante\Desktop\ai-first-workspace-template-main\Website
npm install
```

### Шаг 5: Запустить проект

```powershell
npm run dev
```

Откройте браузер: **http://localhost:3000**

---

## ❌ Если что-то не работает

### Проблема: "node не является внутренней или внешней командой"

**Решение:**
1. Перезапустите терминал (закройте и откройте заново)
2. Если не помогло - переустановите Node.js и убедитесь, что выбрана опция "Add to PATH"

### Проблема: Ошибки при npm install

**Решение:**
1. Проверьте, что Node.js установлен: `node --version`
2. Попробуйте удалить папку `node_modules` (если она есть) и запустить `npm install` снова
3. Используйте Command Prompt вместо PowerShell
