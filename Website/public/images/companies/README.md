# Логотипы компаний

Эта папка содержит логотипы компаний, которые интервьюируют кандидатов Go Offer.

## Требования к логотипам

- **Формат**: SVG (предпочтительно) или PNG с прозрачным фоном
- **Размер**: Рекомендуется 200x100px или больше (будет автоматически масштабироваться)
- **Стиль**: Логотипы должны быть в светлом/темном варианте, подходящем для сайта
- **Названия файлов**: Используйте kebab-case (например: `tiktok.svg`, `capital-one.svg`)

## Необходимые логотипы

1. `tiktok.svg` - TikTok
2. `microsoft.svg` - Microsoft
3. `meta.svg` - Meta (Facebook)
4. `amazon.svg` - Amazon
5. `capital-one.svg` - Capital One
6. `disney.svg` - Walt Disney
7. `sony.svg` - Sony
8. `epam.svg` - EPAM
9. `google.svg` - Google
10. `openai.svg` - OpenAI

## Где найти логотипы

- [Simple Icons](https://simpleicons.org/) - бесплатные SVG логотипы компаний
- [Logos Download](https://logos-download.com/) - официальные логотипы
- Официальные сайты компаний (раздел Media/Press)

## Примечание

Все логотипы будут автоматически масштабироваться до размера 112x56px (на десктопе) и 96x48px (на мобильных) с сохранением пропорций.
